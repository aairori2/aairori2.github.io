# My Personal Website

Welcome to my personal website repository! This is a GitHub Pages site hosted at [aairori2.github.io](https://aairori2.github.io).

## About

This repository contains the source code for my personal website, where I share information about myself, my projects, and my professional interests.

## Contents

- **index.html** - Main website page with personal information and content
- **profile.jpg** - Profile picture featured on the website

## Getting Started

This site is built with HTML and is automatically deployed to GitHub Pages whenever changes are pushed to the `main` branch.

### Local Development

To view the website locally:

1. Clone this repository:
   ```bash
   git clone https://github.com/aairori2/aairori2.github.io.git
   cd aairori2.github.io
   ```

2. Open `index.html` in your web browser, or use a local server:
   ```bash
   # Using Python 3
   python -m http.server 8000
   
   # Using Node.js (if you have http-server installed)
   http-server
   ```

3. Visit `http://localhost:8000` in your browser

## Deployment

This site is automatically deployed to GitHub Pages. Any changes pushed to the `main` branch will be live at [aairori2.github.io](https://aairori2.github.io) within a few seconds.

## Customization

To customize this website:

1. Edit `index.html` to update your content
2. Replace `profile.jpg` with your own profile picture
3. Commit and push your changes to `main`
